# PulseBoard Pro

A polished SaaS dashboard starter built with **Next.js 15**, **TypeScript**, **Tailwind CSS**, **Prisma**, **Recharts**, and a clean component structure.

## What is included

- Premium landing page designed to sell the product
- Dashboard shell with sidebar, topbar, and theme support
- Overview, Projects, Tasks, Analytics, Team, and Settings pages
- Login and register screens with product-grade layout
- Reusable UI components and mock data
- Prisma schema starter for a real SaaS backend

## Project structure

- `app/` — Next.js App Router pages and layouts
- `components/` — dashboard UI blocks
- `lib/` — utility helpers, types, and mock data
- `prisma/` — database schema starter

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm start
```
